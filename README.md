# Excel Sales Automation Template

## Overview

This project demonstrates an automated Excel-based solution for structured sales data entry.

The template helps standardize the input process, reduce errors, and simplify further analysis.

## Features

- Dropdown lists for manager, client, product, and product group
- Product search functionality
- Automated price and discount calculation
- Data validation to prevent incorrect input
- Named ranges for scalable and maintainable structure

## How to Use

1. Select a manager  
2. Select a client  
3. Select a product group (optional)  
4. Select a product:
   - Use product group to filter  
   - Or search by product name  
5. Enter quantity  

All calculations are performed automatically.

## Structure

Main sheet:
- Data input form

Hidden sheets:
- Managers
- Clients
- Products
- Product groups
- Price list

## Flexibility

The template can be easily adapted:
- New columns can be added based on business needs
- Lists and pricing can be updated without changing the structure

## Tools Used

- Microsoft Excel
